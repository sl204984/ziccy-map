    enterDrawMode

    enterDrawMode
    (
    • type
    • options
    )

    进入绘制模式。绘制模式下，地图会被锁死，

    Parameters:

    nametypeflagdescription
    type String

    绘制类型，可选值如下：

    • null 自由绘制
    • "circle" 画圆
    • "rectangle" 矩形框
    options Object

    绘制选项

    • autoExit

    Example:

        _map.enterDrawMode("rectangle", {
          autoExit: function(map) {
               var poly = map.exitDrawMode();
             map.addOverlay(poly);
        },
 });

    Events

    init

    地图初始化事件，仅在地图对象被创建后触发一次

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="init"; event.target = map;

    move

    地图移动事件

    • 无论以何种方式，使地图发生中心点改变时均会触发。

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="move"; event.target = map;

    movebegin

    地图移动开始事件

    • 当地图移动API调用条件下，移动开始时触发；
    • 当鼠标拖动地图移动条件下，鼠标左键按下后，第一次发生位置移动时触发；
    • 当移动设备上，触摸拖动地图时，手指按下屏幕后，第一次发生识别位置改变时触发。

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="movebegin"; event.target = map;

    moveend

    地图移动结束事件。

    • 当地图移动API调用条件下，移动完成，地图中心点到达目标经纬度时触发；
    • 当鼠标拖动地图移动条件下，鼠标左键松开时触发；
    • 当移动设备上，触摸拖动地图时，手离开屏幕时触发。

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="moveend"; event.target = map;

    ready

    地图初始化就绪事件，仅在地图对象被创建后，地图开始正常的渲染显示时触发一次

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="ready"; event.target = map;

    resize

    地图尺寸改变事件

    • 浏览器窗口尺寸发生改变，地图尺寸逐之改变时触发；
    • 调用地图API改变其尺寸后触发；

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="resize"; event.target = map;

    tileready

    地图数据就位事件

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="tileready"; event.target = map;

    zoom

    地图缩放事件

    • 无论以何种方式，使地图发生缩放比例、级别改变时均会触发。

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="zoom"; event.target = map;

    zoombegin

    地图缩放开始事件

    • 当地图缩放API调用条件下，缩放开始时触发；
    • 当点击地图上的缩放按钮条件下，鼠标点击（长按不予响应）后松开时触发；
    • 当鼠标滚轮滚动缩放动地图条件下，在上次滚动结束后，鼠标滚轮第一次发生滚动时触发；
    • 当移动设备或支持多点触摸的设备上，两个（及以上）手指触摸后，第一次手指聚拢或分开的动作被识别时触发。

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="zoombegin"; event.target = map;

    zoomend

    地图缩放结束事件

    • 当地图缩放API调用条件下，缩放级别、比例到达目标级别、比例，缩放动作结束时触发；
    • 当点击地图上的缩放按钮条件下，鼠标点击后，地图缩放动画结束时触发；
    • 当鼠标滚轮滚动缩放动地图条件下，在上次滚动发生后200毫秒内没有再发生滚动时触发；
    • 当移动设备或支持多点触摸的设备上，两个（及以上）手指触摸后，上次手指聚拢或分开的动作的被识后200毫秒内没有再发生类似的操作时触发。

    Parameters:

    nametypeflagdescription
    event Object

    事件对象，event.type="zoomend"; event.target = map;
